Інтерактивне навчання про атмосферу для дітей

 Опис проекту

Цей проект містить серію інтерактивних веб-сторінок для навчання дітей основам атмосферних явищ:

- Зміна температури повітря вдень
- Утворення хмар
- Вітер та його причини
- Атмосферний тиск
- Види опадів
- Шари атмосфери
- Парниковий ефект
